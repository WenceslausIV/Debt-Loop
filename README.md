# Debt-Loop

3D rogue like puzzle game(made with Unity)

Download here https://www.dropbox.com/s/9j9jlqgvm8gmd68/Debt%20Loop.zip?dl=0

WASD - move
T - teleport to dungeon
I - open inventory
